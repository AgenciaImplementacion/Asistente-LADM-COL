Installation guide for plugin¶

This section describes the process to perform the installation of the QGIS plugin assistant LADM_COL.

Installation process¶

  • It is necessary to have QGIS 3 version Software installed, we recommend to use the weekly version for now until the 3.4 LTS version has been released, to obtain this go to QGIS.

  • The process can be seen graphically in the following gif:

  • Make sure you have the last versión of the QgisModelBaker plugin, the Assistant LADM_COL depends on this plugin to work, otherwise this message will appear:

  • If you get an error, you can install the QgisModelBaker plugin like in the following gif:

Subscribe to the releases of Asistente-LADM_COL¶

  1. Enter to LINK.

  2. Search Asistente-LADM_COL.

  3. Select enter email.

  4. sign in or sign up the email and the passwaord.

  5. Ready, you are suscribed to the releases of Asistente-LADM_COL.
