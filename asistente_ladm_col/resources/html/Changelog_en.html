Changelog for version 2.99.0

  • Integración con el Sistema de Transición. Autenticación y gestión de tareas.
  • Soporte de roles y generación de interfaz de usuario para cada rol.
  • Actualización de modelos para el catastro multipropósito. Esta versión del Asistente LADM_COL soporta los modelos versión 2.9.6.
  • ETL para generar insumos catastrales a partir de datos del IGAC (fuente Cobol).
  • Nuevo reporte de Omisiones y Comisiones.
  • Soporte de ili2db4 (llaves foráneas para relaciones dominio-clase, soporte de relaciones 1:m con atributos, entre otras novedades).
