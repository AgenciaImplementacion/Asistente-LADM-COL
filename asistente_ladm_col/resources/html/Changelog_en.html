Changelog for version 2.99.1

  • Integración con el Sistema de Transición:
    • Iniciar, cancelar y finalizar tarea desde el Asistente LADM_COL.
    • Tareas de generación de insumos catastrales e Integración asistida (soporte parcial).
  • Identificación de novedades entre datos del barrido predial y datos de insumos.
  • Soporte de GeoPackage como base de datos para manejar datos LADM-COL (exceptuando validaciones de calidad, consultas y reportes).
  • Usabilidad: Instalación automática de prerrequisitos: incluye los complementos Model Baker y Map Swipe Tool, modelos LADM-COL v2.9.6 y Java v1.8.
  • Mejoras en pruebas:
    • 1 prueba de interfaz semi-asistida (nueva).
    • 150 pruebas unitarias automáticas.
