Cambios en la version 3.1.0

  • Soporte para el código EPSG oficial de la proyección "Origen Nacional" (CTM12); esto es: EPSG:9377.
  • Soporte de importación masiva de datos (Modelo ETL) para QGIS v3.14 (Fix #350).
  • Soporte de reproyección (automática) a EPSG:9377 cuando se importan datos masivamente desde otros sistemas de referencia (Fix #337).
  • Valores predeterminados para dominios. Por ejemplo, al crear un interesado, en el campo "Tipo" se mostrará el valor "Interesado Natural" por defecto.
