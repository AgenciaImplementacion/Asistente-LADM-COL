Guía de Instalación para el plugin¶

Esta sección describe el proceso para realizar la instalación del plugin de QGIS Asistente LADM_COL.

Proceso de instalación¶

  • Es necesario tener el Software QGIS versión 3 instalado, recomendamos usar la versión semanal hasta que se haya lanzado la versión 3.4 LTS, para obtener este diríjete a QGIS.

  • El proceso puede ser observado graficamete en el siguiente GIF:

  • Asegúrate de tener la última versión del plugin QgisModelBaker, el Asistente LADM_COL depende de este plugin para funcionar, de lo contrario este mensaje aparecerá:

  • Si tienes un error, puedes instalar el plugin QgisModelBaker como en el siguiente gif:

Suscríbete a los lanzamientos del Asistente-LADM_COL¶

  1. Enter to LINK.

  2. Busque Asistente-LADM_COL.

  3. Seleccione ingrese correo electronico.

  4. Inicia sesión o regístrate el correo electrónico y la contraseña.

  5. Listo, estás suscrito a los lanzamientos del Asistente-LADM_COL.
