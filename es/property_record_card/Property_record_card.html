Ficha Predial¶

Crear Ficha Predial¶

  1. Ingresando datos manualmente en un formulario

Elige esta opción si deseas agregar una Ficha Predial con un formulario.

Ficha Predial se crea específicamente para almacena la información recogida para los predios.

  1. Desde otra capa de QGIS (definiendo un mapeo de campos)

Elige esta opción para abrir una ventana que te permite importar datos desde una tabla fuente hacia la tabla predio_ficha de LADM_COL.

Si la estructura de campos de las tablas de entrada y salida difiere, puedes definir un mapeo para transformar campos y establecer correspondencias entre ellos.

Para usar esta función revisa este ENLACE.
